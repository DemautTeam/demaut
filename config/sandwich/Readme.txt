-------------------
Projet Sandwich : https://github.com/damianszczepanik/cucumber-sandwich

-> Permet de générer des jolis rapports à partir des rapports JSon générés par Cucumber

-------------------
Utilisation

- lancer les tests BDD via les Runner qui doivent produire des extraits json sous demaut-bdd/target/bdd-mock-json et bdd-data-json
- lancer sandwich.sh pour créer les jolis rapports qui sont disponibles sous : demaut-bdd/target/bdd-mock-html/cucumber-html-reports et sous bdd-data-html/cucumber-html-reports
